安裝說明
1打開「100_Run」Google 試算表 → 擴充功能 → Apps Script

3部署 → 新增部署 → 網頁應用程式 → 執行者：我，存取：所有人

Windows Chrome：網址列右側安裝圖示 → 安裝
Android Chrome：選單 → 新增至主畫面
